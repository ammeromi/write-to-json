# write-to-json
A script that fetches from [JSON placeholder](https://jsonplaceholder.typicode.com/posts)  posts and write the response to a json file

# Dependencies
What you need to run this project:
- Node.js v10+
- Yarn v1.22.5+

# How to run
## 1. Install dependencies

```
$ yarn
```

## 2. Run script
```
$ node index.js
```
